## Probix — AI Operations Platform

Live at probixai.com. Real clients, real calls, real revenue.

Probix handles the full customer lifecycle for HVAC and home service companies. When a customer calls, an AI voice agent named Sarah picks up, figures out what they need, and books the job. If they don't call back, an automated SMS sequence follows up. After the job is done, a review request goes out. All of it runs without anyone touching it.

**What's inside:**

- Sarah, an AI voice agent built on LiveKit with Telnyx SIP for call handling, Deepgram for speech recognition, Cartesia for voice output, and Groq's Llama model for reasoning
- A webhook automation engine covering missed calls, estimate follow-ups, reactivation campaigns, and post-job review requests
- A full admin dashboard for estimates, bookings, contacts, invoices, and technician management
- A remote MCP server with 25 tools that gives Claude AI full access to all business operations from any device

**Built with:** Next.js 14, Python, Supabase, Vercel, LiveKit, Telnyx, Groq, Deepgram, Cartesia, Tailwind CSS
